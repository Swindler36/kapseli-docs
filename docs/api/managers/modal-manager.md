Modal Manager provides an API to create modals in several types on your app.

## Events {#events}
* `modal:open`
* `model:close`
* `modal:open:{modalID}`
* `model:close:{modalID}`

## Functions {#functions}
* add
* remove
* get
* getOpen
* setOpen
* getModals

### Modal Object Functions {#modal-object-functions}
* open
* close
* getTitle
* setTitle
* getContent
* setContent
