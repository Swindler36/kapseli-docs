Data Manager is a functionality class to manipulate data to use storage manager.

## Events {#events}
* `data:get` => get()
* `data:set` => set()
* `data:update` => update()
* `data:delete` => delete()
* `data:dump` => dump()
* `data:load` => load()

## Functions {#functions}
* get
* set
* update
* delete
* has
* sort
* filter
* dump `to JSON`
* load `from JSON`
